# Live Auction House

A real-time auction platform built with:

- React + Vite + Tailwind CSS
- Node.js + Express
- MongoDB
- Socket.io (Real-time bidding)

## Features
- Live bidding system
- Auction timer (sniper logic)
- Email notification for winner
- Admin dashboard with analytics

## Run Project

### Frontend
npm install
npm run dev

### Backend
npm install
npm run dev
